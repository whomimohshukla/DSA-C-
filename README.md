# C++ DSA Daily Preparation

This repository is for **daily DSA (Data Structures and Algorithms) practice in C++**.

## Description

In this repo, we will focus on:
- Daily problem solving
- Building strong DSA concepts
- Improving coding speed and logic for interviews and placements
- Writing clean and optimized C++ solutions

## Repository Structure

- `Array/` -> Array-based DSA problems
  - `secondLargest.cpp` -> Find the second largest element problem

As practice grows, more folders will be added (like `LinkedList/`, `Stack/`, `Queue/`, `Tree/`, `Graph/`, etc.).

## Goal

The goal is consistent daily preparation to become confident in solving DSA problems using C++.

## How to Run a File

Example (for `secondLargest.cpp`):

```bash
g++ Array/secondLargest.cpp -o secondLargest
./secondLargest
```

## Note

This repository will be updated regularly with new daily DSA problems and solutions.
